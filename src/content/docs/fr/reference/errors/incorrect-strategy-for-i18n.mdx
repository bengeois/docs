> **IncorrectStrategyForI18n**: La fonction `FUNCTION_NAME` ne peut être utilisée que lorsque `i18n.routing.strategy` est défini sur `"manual"`.

## Qu'est-ce qui n'a pas fonctionné ?
Certaines fonctions d'internationalisation ne sont disponibles que lorsque le routage i18n d'Astro est désactivé par le paramètre de configuration `i18n.routing` défini sur `"manual"`.
